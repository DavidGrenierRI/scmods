# SCMODS - Consulting Firm Management System

A Django-based project management and scheduling system for consulting firms.

## Features (Planned)

- Hierarchical task management with infinite nesting
- Task dependencies and prerequisites
- Intelligent scheduling with time estimates
- Multi-user system (admin + client access)
- Time tracking with timers
- Document repository
- Automated scheduling worker
- Client-facing status reports

## Tech Stack

- Django + Django REST Framework
- PostgreSQL
- Redis
- React (frontend)
- Docker & Docker Compose

## Development Setup

Prerequisites:
- Docker Desktop

Getting started:
```bash
# Clone the repository
git clone <your-repo-url>

# Start the development environment
docker compose up

# Access the application at http://localhost:8000
```
